# Privacy Policy for SDDC-Core Automation

**Last Updated:** March 16, 2026

## Overview
This Privacy Policy describes how the **SDDC-Core Automation** app (the "Application") handles data. This Application is a private, non-commercial tool developed by **sonydogg** for personal use within a home lab environment.

## Data Collection and Usage
The Application uses Google OAuth strictly to facilitate a "Human-in-the-Loop" approval process via Gmail. 

* **User Data Access:** The Application only accesses the minimum necessary Gmail scopes to send approval requests to the developer's own email address.
* **Storage:** No user data from Google services is stored on external servers. All processing occurs within a private, self-hosted n8n instance.
* **Sensitive Information:** The Application does not collect, store, or share any sensitive personal information, financial data, or PII (Personally Identifiable Information).

## Data Sharing and Disclosure
* **Third-Party Sharing:** We do not sell, trade, or otherwise transfer your information to outside parties. 
* **Service Providers:** Data is only transmitted between the user's controlled GitHub repository, the private n8n instance, and the user's Atlassian Confluence workspace.

## Security
We implement industry-standard security measures, including Cloudflare tunneling and encrypted credentials, to maintain the safety of the automation pipeline.

## Contact
If you have any questions regarding this personal project, you may contact the developer via the GitHub repository issues.
