# ADR-012: Initial Packer Template for Windows Image Factory

**Status:** Proposed (2026-03-16)

**Context:**
To move away from SCCM and "Golden Images", we require a declarative configuration that defines how a Windows instance is built from a raw ISO. This file serves as the blueprint for the "Stones" Image Factory.

**Technical Blueprint (Packer HCL):**
```hcl
# This file should be stored in Perforce: //depot/images/windows/ws2022.pkr.hcl

packer {
  required_plugins {
    vmware = {
      version = ">= 1.0.0"
      source  = "[github.com/hashicorp/vmware](https://github.com/hashicorp/vmware)"
    }
  }
}

source "vmware-iso" "sddc-windows" {
  # The Librarian provides the source from the Water stone
  iso_url          = "/mnt/stones/water/isos/windows_server_2022.iso"
  iso_checksum     = "sha256:your-checksum-here"
  
  vm_name          = "SDDC-Win-Base"
  guest_os_type    = "windows2019srv-64"
  
  # Resource allocation for the build
  cpus             = 4
  memory           = 8192
  disk_size        = 61440
  
  # Headless build for SDDC Core efficiency
  headless         = true
  
  # Injecting the logic via the Wind stone
  floppy_files     = ["./scripts/autounattend.xml"]
  
  ssh_username     = "admin"
  ssh_password     = "password"
  ssh_timeout      = "4h"
}

build {
  sources = ["source.vmware-iso.sddc-windows"]

  # Post-install logic: This is where we replace Ninite/SCCM
  provisioner "powershell" {
    scripts = [
      "./scripts/enable-ssh.ps1",
      "./scripts/install-winget.ps1"
    ]
  }
}
